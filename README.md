# internship-scam-review-portal
Java Servlet JSP Internship &amp; Scam Review Portal using Oracle Database

## Features

- User Registration
- User Login & Logout
- Session Management
- Add Internship Companies
- View Internship Companies
- Add Reviews
- View Reviews
- Report Scam Companies
- View Scam Reports

## Technologies Used

- Java
- Servlets
- JSP
- JDBC
- Oracle Database
- HTML
- CSS
- Bootstrap 5
- Apache Tomcat

## Database Tables

- users
- companies
- reviews
- scam_reports
